# EZ_SmartAssistant_Project
EZ SmartAssistant is a Python-based AI tool that helps users analyze and visualize data efficiently. 
It provides an interactive dashboard using Streamlit to explore datasets, generate insights, and export summary reports.
## Technologies Used
- Python (pandas, numpy)
- Streamlit for UI
- Matplotlib / Plotly for visualization
- PDF report generation

## Installation & Setup
1. Clone the repo:
   git clone https://github.com/priyasingh814/EZ_SmartAssistant_Project.git
2. Install requirements:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
![Dashboard Screenshot](https://github.com/priyasingh814/EZ_SmartAssistant_Project/blob/main/Screenshot%202025-07-13%20185159.png)

## Features
- Upload PDF files to extract data
- Summarize PDFs automatically
- Answer questions from PDF content
- Interactive dashboard using Streamlit
